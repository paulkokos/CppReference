#```count() ``` , ```count_if()```
* ```count()```<br>
Επιστρέφει τον αριθμό των στοιχείων που ικανοποιούν συγκεκριμένα κριτήρια.<br>
Μετραει τα στοιχεια τα οποια ειναι ισα με την τιμη.<br>
* ```count_if()``` <br>
Επιστρέφει τον αριθμό των στοιχείων που ικανοποιούν συγκεκριμένα κριτήρια.<br>
Μετραει τα στοιχεια τα οποια η δηλωση ```p``` επιστρεφει ```true```.
